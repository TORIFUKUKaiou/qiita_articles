# @tamanugi さん

```elixir
Stream.cycle('元気ですかーーーーッ！ 元気があればなんでもできる')
|> Stream.map(&([IO.ANSI.color(Enum.random(0..255)), &1]))
|> Stream.each(&IO.write/1)
|> Stream.run
```

Thanks!

# @Mnishiguchi さん

```elixir
for c <- Stream.cycle('元気ですかーーーーッ！ 元気があればなんでもできる') do
  IO.write([IO.ANSI.color(Enum.random(0..255)), c])
end
```

ありがとーーーーう！
