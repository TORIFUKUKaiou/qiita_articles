<details><summary>私の解答</summary>

_問題文を読んでいることを前提にひとこと解説をしておきます。_

野球のスコアボードを題材にした問題です。
リストの要素の足し算をすればよいわけで、[Enum.sum/1](https://hexdocs.pm/elixir/Enum.html#sum/1)関数が大活躍です。


```elixir
defmodule Main do
  def main do
    a_list =
      IO.read(:line) |> String.trim() |> String.split(" ") |> Enum.map(&String.to_integer/1)
    b_list =
      IO.read(:line) |> String.trim() |> String.split(" ") |> Enum.map(&String.to_integer/1)
    
    solve(a_list, b_list)
    |> IO.puts()
  end
  
  def solve(a_list, b_list) do
    Enum.sum(a_list) - Enum.sum(b_list) + 1
  end
end
```
