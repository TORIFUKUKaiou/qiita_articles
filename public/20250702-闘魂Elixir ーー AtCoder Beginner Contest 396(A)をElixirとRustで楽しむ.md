## Elixir

Elixirを使った私の解答です。


<details><summary>私の解答(Elixir)</summary>

_問題文を読んでいらっしゃることを前提にひとこと解説をしておきます。_


Enum.reduce_while/3 で解きました。これはお手の物です。  

```elixir
defmodule Main do
  def main do
    IO.read(:line)
    list =
      IO.read(:line) |> String.trim() |> String.split(" ") |> Enum.map(&String.to_integer/1)
    
    solve(list)
    |> IO.puts()
  end

  def solve(list) do
    list
    |> Enum.reduce_while({nil, 0}, fn
      a, {nil, 0} -> {:cont, {a, 1}}
      a, {a, 2} -> {:halt, {a, 3}}
      a, {a, 1} -> {:cont, {a, 2}}
      a, {_, _} -> {:cont, {a, 1}}
    end)
    |> elem(1)
    |> Kernel.==(3)
    |> if(do: "Yes", else: "No")
  end
end
```

## Rust

RustはAI先生のお力をお借りして、Elixirのコードを置き換えてもらいました。
私は、Rustを勉強中です。万年勉強中です。闘魂にゴールはない。いつまでも挑戦中です。

<details><summary>私の解答(Rust)</summary>

```rust
use std::io::{self, BufRead};

fn main() {
    let stdin = io::stdin();
    let mut lines = stdin.lock().lines();

    // 1行目を無視（IO.read(:line) に相当）
    let _ = lines.next();

    // 2行目：数字のリストを取得
    let list: Vec<i32> = lines
        .next()
        .unwrap()
        .unwrap()
        .trim()
        .split_whitespace()
        .map(|s| s.parse().unwrap())
        .collect();

    // 判定と出力
    println!("{}", solve(&list));
}

fn solve(list: &[i32]) -> &'static str {
    let mut prev: Option<i32> = None;
    let mut count = 0;

    for &num in list {
        match prev {
            Some(p) if p == num => {
                count += 1;
                if count == 3 {
                    return "Yes";
                }
            }
            _ => {
                prev = Some(num);
                count = 1;
            }
        }
    }

    "No"
}
```

</details>
