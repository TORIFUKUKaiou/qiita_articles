## Elixir

Elixirを使った私の解答です。


<details><summary>私の解答(Elixir)</summary>

_問題文を読んでいらっしゃることを前提にひとこと解説をしておきます。_

文字列をN個読み込んで、文字列の長さの昇順に並び替えて、最後に連結しました。  


```elixir
defmodule Main do
  def main do
    n = IO.read(:line) |> String.trim() |> String.to_integer()
    list = for _ <- 1..n do
      IO.read(:line)
      |> String.trim()
    end

    solve(list)
    |> IO.puts()
  end

  def solve(list) do
    list
    |> Enum.sort_by(& String.length(&1))
    |> Enum.join("")
  end
end
```



</details>

## Rust

RustはAI先生のお力をお借りして、Elixirのコードを置き換えてもらいました。
私は、Rustを勉強中です。万年勉強中です。闘魂にゴールはない。いつまでも挑戦中です。

<details><summary>私の解答(Rust)</summary>

```rust
use std::io::{self, BufRead};

fn main() {
    let stdin = io::stdin();
    let mut lines = stdin.lock().lines();

    // 1行目の入力を整数として読み取る
    let n: usize = lines.next().unwrap().unwrap().trim().parse().unwrap();

    // n行の文字列を読み取る
    let mut list: Vec<String> = Vec::with_capacity(n);
    for _ in 0..n {
        if let Some(Ok(line)) = lines.next() {
            list.push(line.trim().to_string());
        }
    }

    // 長さ順にソートして連結
    list.sort_by_key(|s| s.len());
    let result = list.concat();

    println!("{}", result);
}
```

</details>
