問題はリンク先をご参照くださいませ。
私の解答を貼っておきます。

プログラミングの基本である「順次」「分岐」「繰り返し」のうち特に「順次」を理解できているのかが問われる問題です。

## Elixir

Elixirを使った私の解答です。


<details><summary>私の解答(Elixir)</summary>

_問題文を読んでいらっしゃることを前提にひとこと解説をしておきます。_

文字列のパターンマッチ一発で解きました。  


```elixir
defmodule Main do
  def main do
    s = IO.read(:line) |> String.trim()

    solve(s)
    |> IO.puts()
  end

  def solve(<<a, ?x, b>>) do
    a = a - ?0
    b = b - ?0

    a * b
  end
end
```



</details>

## Rust

RustはAI先生のお力をお借りして、Elixirのコードを置き換えてもらいました。
私は、Rustを勉強中です。万年勉強中です。闘魂にゴールはない。いつまでも挑戦中です。

<details><summary>私の解答(Rust)</summary>

```rust
use std::io::{self, Write};

fn main() {
    // 入力読み取り
    let mut input = String::new();
    io::stdin().read_line(&mut input).unwrap();
    let s = input.trim();

    // 期待通り "2x8" 形式なら処理
    let chars: Vec<char> = s.chars().collect();
    if chars.len() == 3 && chars[1] == 'x' {
        let a = chars[0].to_digit(10).unwrap();
        let b = chars[2].to_digit(10).unwrap();
        println!("{}", a * b);
    } else {
        eprintln!("Unexpected input format");
    }
}
```

</details>
